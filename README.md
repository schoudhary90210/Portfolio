# SCHOUDHARY CAPITAL

**Siddhant Choudhary** — 3D Quant Trading Floor Portfolio

**Live:** [siddhantchoudhary.vercel.app](https://siddhantchoudhary.vercel.app)

An isometric 3D trading floor built with React Three Fiber featuring 6 Bloomberg terminal workstations, each representing a project. Dark room, green phosphor glow, CRT screens, phantom knight sentinel.

## Tech Stack

- React + Vite
- Three.js / React Three Fiber / Drei
- GSAP (camera animations)
- Post-processing (Bloom, Vignette, Noise)

## Run Locally

```bash
npm install
npm run dev
```
